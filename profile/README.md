### 组织历史：
- 2020年5月，水滴实验室正式成立。
- 2020年5月，王克硕和谈致远正式成为该组织的成员。
- 2020年5月，QQ群"水滴实验室"创建。
- 2020年5月，成员王克硕为该组织的群聊替换图标，从此，该组织的图标正式确定。
- 2020年5月1日，该组织的第一个软件"PowerSettings V.5.0.0"完工。

- 2023年6月16日,谭景元和梁皓文正式成为该组织的成员。
- 2024年6月16日，“水滴实验室”更名为“冰软工作室”。
